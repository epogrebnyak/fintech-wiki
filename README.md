# global-fintech-wiki

We are building an LLM-powered wiki covering **fintech** (payments, neobanks, wealthtech) and **digital infrastructure** (blockchain and tokenization).

## Tooling

- **Wiki management**: [llm_wiki](https://github.com/nashsu/llm_wiki) — use LLMs to generate and maintain wiki content from source documents.
- **Publishing**: [Quartz](https://quartz.jzhao.xyz/) — publish the wiki as a static site to GitHub Pages.
